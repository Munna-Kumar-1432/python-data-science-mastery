What is Machine Learning? — Notes
Formal definition
Machine learning is a field of computer science that uses statistical techniques to give computer systems the ability to 'learn' from data, without being explicitly programmed for each task.
3 key concepts unpacked
Core concept
The system learns from data — it improves its performance automatically as it is exposed to more examples.
Data-driven
Key difference from traditional programming
You don't write rules manually for every scenario. The algorithm figures out the rules on its own from patterns in data.
No explicit rules
What "statistical techniques" means
ML uses probability & math to find patterns — not guessing. It calculates the most likely answer based on what it has seen.
Math-based
How the process works — step by step
1
Provide data to the algorithm — You feed it thousands of labelled examples. E.g. 10,000 emails marked "spam" or "not spam".
2
Algorithm identifies patterns — It finds relationships between the input (email text) and the output (spam/not spam). E.g. "emails with 'Win money!' tend to be spam."
3
Use learned patterns on new inputs — When a brand-new email arrives, the model applies those patterns to predict whether it's spam — without you writing any rules.
Real-world analogy
Think of it like teaching a child
You don't give a child a rulebook to identify a dog. You show them 100 pictures of dogs, and over time they learn the pattern — four legs, fur, tail. ML works exactly the same way: show the system enough examples, and it learns to recognize patterns on its own.
Traditional programming vs machine learning
Traditional programming
Data + Rules → Output
You write the rules. The computer follows them to produce results.
Machine learning
Data + Output → Rules
You give data and answers. The computer figures out the rules itself.
